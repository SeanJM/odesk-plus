odesk-plus
==========

A chrome extension which improves usability for the oDesk.com web application

How to install
==============
1. Open Chrome
2. Customize Chrome (the settings panel)
3. Extensions
4. 'Load Unpacked Extension'
5. Navigate to git repository and load
6. Celebrate
